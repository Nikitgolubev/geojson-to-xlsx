# backups

Здесь хранится синхронизируемая копия данных (города и зоны) из вкладки
«Обновление данных»:

- `manifest.json` — мета актуальности: версия, время, dataHash, счётчики.
- `latest.json` — полный снимок данных для восстановления.

Файлы создаёт/обновляет программа polygons через GitHub Contents API.
Версионирование — поле `version` + история git.
