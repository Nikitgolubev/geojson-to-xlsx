Конвертер GeoJSON → Excel (.xlsx / ZIP)

Выберите один или несколько файлов .geojson/.json.
• Если выбран один файл — будет скачан один .xlsx.
• Если выбрано несколько файлов — будет скачан ZIP со множеством .xlsx.

Можно выделить несколько файлов сразу (Ctrl/Shift + клик).
Используется только если выбрано больше одного файла. Если пусто, используется имя по умолчанию.
Лог / прогресс
Кнопка «Очистить выбор и результаты» не очищает лог — история использования сохраняется в этом окне.
